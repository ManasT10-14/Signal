# PCP — What to Remember
### Perception × Context × Permission

---

## The One Thing

Every human behavior change runs through the same sequence:

> **Shift how someone sees the situation → redefine the rules of that situation → the behavior you want becomes the obvious thing to do.**

You don't persuade people. You engineer the conditions where they persuade themselves.

---

## The Three Parts

### 1. PERCEPTION — Control What They See

You are always inside a frame. There is no neutral perception. Whoever sets the frame first controls the interaction.

**The 14 frames in plain english:**

| Frame | The Simple Version |
|-------|--------------------|
| **Gain vs. Loss** | Losses hurt 2x more than gains feel good. If you need action — use loss. If you need comfort — use gain. |
| **Identity** | Don't compliment them directly. Name a negative group they'd hate to belong to. They silently commit to being the opposite. |
| **Temporal** | Stuck in the past? Pull them to the future. All talk, no action? Pull them to the present. Overwhelmed? Pull them to the past where they survived. |
| **Role** | Define your own role before they do. Intern → consultant. Founder begging → offer of access. Whoever defines the roles first wins. |
| **Causal** | Same event, different cause = different emotion. "I'm broken" → "I have a pattern. Patterns can change." |
| **Scale** | Zoom out when you're anxious. One bad day isn't a trend. What does the data say over 3 months? |
| **Comparison** | Always anchor first at your number/position. They negotiate from your reference point, not theirs. |
| **Moral/Values** | Make it about character and they can't say no without feeling like a bad person. Make it practical when you want to de-escalate. |
| **Power/Status** | "Thank you so much for your time" puts you below them instantly. Speak like a peer. Act like you're both evaluating each other. |
| **Scarcity** | Fake scarcity is detected and repels. Real scarcity (a genuinely full life) is attractive. |
| **Certainty** | Be certain on the problem. Be honest about uncertainty on the solution. Total certainty reads as arrogance or naivety. |
| **Collaborative** | Open with a shared enemy or shared goal before any negotiation. Converts competition into collaboration before it starts. |
| **Narrative** | Figure out what story they think they're in. Hero? Underdog? Mentor? Their decisions follow the story's logic. |
| **Meta** | Define how the conversation works before it starts. "This is brainstorming, nothing said is a commitment" changes everything people are willing to say. |

**The 12 ways to shift a frame:**

| Technique | One Line |
|-----------|----------|
| **Content Reframe** | Same facts, different meaning. "She's not replying fast" → "She's busy AND still choosing to reply." |
| **Context Reframe** | That weakness is a strength in a different setting. Overthinking = due diligence superpower. |
| **Temporal Reframe** | Failure at 1 year = learning at 10 years. Change the window of evaluation. |
| **Chunk Up** | Go to the shared goal. Conflict dissolves when both people want the same thing and only disagree on how. |
| **Chunk Down** | Force specifics. Fear lives in vague abstractions. "Name one example" breaks it. |
| **Outcome Reframe** | Stop analyzing the problem. "What would the solution look like?" redirects energy forward. |
| **Counter-Example** | One exception breaks any always/never belief. Find it. |
| **Intention Reframe** | "Your intention was good. The impact was different. How do we fix impact without losing intention?" |
| **Pre-Frame** | Set the frame before you say anything important. Negative contrast → positive declaration in the first 30 seconds. |
| **Presupposition** | Embed the assumption in the question. "When you implement this, what support do you need?" assumes it's happening. |
| **As-If** | Remove the obstacle hypothetically. "If that wasn't an issue, what would you do?" Once they answer, the obstacle shrinks. |
| **Surfacing** | Name the pattern out loud. "I notice we do this dance where I push and you pull back. Can we talk about the dance?" Named patterns lose their power. |

### 2. CONTEXT — Control the Rules of the Room

People don't decide what's appropriate. Context decides for them.

You get naked in a shower, not in an office. Same person, totally different behavior — because context changed the rules.

**Five levers of context:**

| Layer | What It Is | Simple Example |
|-------|-----------|----------------|
| **Environment** | The physical space | Coffee shop = exploratory. Boardroom = transactional. Choose deliberately. |
| **Relational** | Your history with them | What you did 3 months ago changes what your actions mean today. |
| **Temporal** | When it happens | A 2am message means something different than a 2pm message. |
| **Social** | What others are doing | "Everyone in this space is moving on this" changes risk tolerance. |
| **Declared** | What you say the context is | "I'm glad we can have a real conversation and not just a pitch" — you just set the rules. |

**Context stacking = when all five align at once.** When they do, the behavior you want feels obvious — not like something you convinced them to do.

---

### 3. PERMISSION — The Behavior That Follows

Permission is not something you ask for. It's what automatically happens when perception and context align.

**Four types:**
- **Behavioral** — "It's okay to do X here." Set by context.
- **Emotional** — "It's okay to feel Y." Set by going there first yourself.
- **Identity** — "It's okay to be the kind of person who Z." Set by showing that people they respect do it.
- **Exit** — "It's okay to leave / say no." Paradox: giving this increases willingness to stay. It signals you're not desperate.

**Self-permission:** The hardest version. Your behavior is caged by the identity you believe you have. Change the identity → change what you allow yourself to do.

## MICRO-COMPLIANCE — How to Get There

### The Core Idea

Small yeses build momentum toward big yeses. Never start with the big ask. Start thin and expand.

Think of it like a wedge. The thin edge goes in first.

**Why it works:**
1. Once someone has said yes a few times, saying no feels inconsistent with who they've been in the conversation
2. A small commitment early makes a large commitment later feel like a natural extension
3. People in motion stay in motion — stopping requires more energy than continuing

### The Five Levels (Go in Order)

| Level | Type | Examples |
|-------|------|---------|
| **L0** | Just being there | They stayed, they're listening, they haven't left |
| **L1** | Tiny physical | "Could you hold this?" / directing their attention |
| **L2** | Easy verbal agreement | "You've probably seen this too, right?" / "Makes sense?" |
| **L3** | Active contribution | "What's your take on this?" / "Walk me through your thinking" |
| **L4** | Identity statement | "Have you always been this way?" — they articulate who they are |
| **L5** | Concrete action | "What's your email?" / "When should we reconnect?" |

By L5 — they've agreed, contributed, declared an identity, and taken action. The big decision now feels consistent with everything they've already done.

### Real Sequences

**Investor meeting:**
Show up (L0) → direct to one slide (L1) → agree on market size (L2) → ask about their portfolio (L3) → ask what drew them to the space (L4) → ask for email to send data room (L5)

**First week at internship:**
Be present (L0) → ask for a doc link (L1) → confirm how things work here (L2) → offer a small observation (L3) → ask why they built the team this way (L4) → present a one-pager at standup (L5)

### Use It On Yourself

Want to build a new habit or break an old one? Don't start with the hard version.

*"Stop checking her profile"* is the thick end. Don't start there.

Start with: notice the urge → set a timer before acting → write what you're feeling → do one alternative thing instead → extend the timer tomorrow.

Same principle. Small compliance with yourself → momentum → behavior change.

## THE FULL SEQUENCE — Any High-Stakes Interaction

1. **Before the conversation** — decide what frame you want, what context you need, what identity commitment you want from them
2. **First 30 seconds** — set the frame. Negative contrast → positive declaration. *"Most people X. I'm glad we can Y."*
3. **First 5-10 minutes** — build the compliance ladder. L0 → L1 → L2 → L3. Small agreements only.
4. **Mid-conversation** — trigger the identity commitment. Negative dissociation or *"Have you always been this way?"*
5. **The ask** — if you've done the above, this moment almost handles itself. Permission has already cascaded.
6. **Close** — convert permission to action. *"What's the next step? When do we reconnect?"*

---

## HOW TO DEFEND AGAINST IT

When something feels off in a conversation, ask:

- *"Who set the frame here — me or them?"*
- *"Have I been making small agreements toward something I haven't consciously decided on?"*
- *"Did I just make an identity statement that now feels like it's boxing me in?"*

**The counter-move:** Name it. *"I notice we've been operating as if X is agreed — I don't think I've actually agreed to that."* Naming the dynamic breaks its hold.

---

## The Three Things to Remember Forever

**1. Whoever sets the frame first controls the interaction.**
Do it in the first 30 seconds or someone else will do it for you.

**2. You don't convince people. You engineer the conditions where they convince themselves.**
The ask should feel like the obvious next step, not a persuasion moment.

**3. Start thin. Everything starts thin.**
Small yes → small yes → small yes → big yes feels natural.
Big yes cold = resistance. Always.
